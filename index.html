⚙ GitHub Actions — Automated Email Setup
Automate daily compliance email alerts using a GitHub Actions workflow:
  1. Export case data as JSON and commit to a GitHub repository
  2. Create .github/workflows/daily-alert.yml with a cron schedule (0 7 * * *)
  3. Write a Node.js script using Nodemailer + SendGrid or Gmail SMTP
  4. Store credentials as GitHub Secrets (EMAIL_USER, EMAIL_PASS, SMTP_HOST)
  5. The workflow runs daily, checks due dates, and dispatches alert emails automatically
